ImageProcessing;

  • C++
  • Binary File I/O
  • Command Line Interface (CLI)

    • This program allows one to edit .tga images. Users can input commands throught the CLI to manipulate images.

    The first step I took when approaching this project was to read the images and learn how to manipulate them before implementing the CLI. Once I familiarized myself with that, implementing the CLI was just a matter of interpreting the command line arguments.

    Interpreting the command line arguments was the hardest part of the project. I had create a system that interpreted the arguments correctly. In the CLI, the user had to input three types of arguments: file names, manipulation methods, and values. Since there were different types of things the user could input, I had to make sure that I was interpreting that specific argument correctly. For example, I would not want to mistake the file name for a method.

    Before this project, working with the terminal was very confusing for me. Though, after building the CLI and creating a makefile, I have gotten a lot more comfortable working with the terminal.
